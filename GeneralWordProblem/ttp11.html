7. General Word Problems

7.2.5 Word Problems with Variables in the Answer Choices

Some GMAT word problems will express quantities as variables rather than as numbers. In these questions, we’ll be tasked with identifying among the answer choices the algebraic expression that represents the value the question asks us to determine.

Consider this mini-example:

Fiona has T batteries, and A of those batteries are alkaline batteries. Which of the following represents the number of Fiona’s batteries that are not alkaline?

(A) T × A

(B) A + T

(C) A – T

(D) TA

(E) T – A

For this question, we must determine how many of Fiona’s batteries are not alkaline, and the answer choices are in terms of the variable T, the total number of batteries Fiona owns, and the variable A, the number of alkaline batteries she owns.

Most importantly, the correct answer will tell us the number of non-alkaline batteries Fiona owns for all possible values of T and A.

In this lesson, we will examine two approaches to handling this question type:

• Applying algebraic techniques

• Testing numbers

We will examine each approach separately when solving the above question.

Please note that you will find that the testing numbers approach typically takes longer than the algebraic approach does. So, in most cases, you should start with the algebraic approach and then turn to testing numbers if the algebraic approach eludes you. That said, we encourage you to use both methods when answering practice questions so that you’ll have all possible tools available to you on test day.

Applying algebraic techniques

When solving a word problem involving variables in the stem and in the answer choices, we can translate the descriptions in the question into algebraic expressions and equations.

For example, the phrase 3 more than x is 5 can be translated as x + 3 = 5. Similarly, the phrase y more than x is z can be translated as x + y = z.

The algebraic approach to solving questions with variables in the answer choices is as follows:

1. Assign a variable to the value we are asked to determine.

2. Use the given information to create one or more equations involving this variable.

3. Solve these equations for the desired variable.

1. Assign a variable to the value we are asked to determine.

2. Use the given information to create one or more equations involving this variable.

3. Solve the equations for the desired variable.

Let’s apply our algebra skills to answer the mini-example we introduced earlier:

Fiona has T batteries, and A of those batteries are alkaline batteries. Which of the following represents the number of Fiona’s batteries that are not alkaline?

(A) T × A

(B) A + T

(C) A – T

(D) TA

(E) T – A

Step 1: Assign a variable to the value we are asked to determine.

Since the question asks us to determine the number of Fiona’s batteries that are not alkaline, let’s let N represent the number of Fiona’s batteries that are not alkaline.

Step 2. Use the given information to create one or more equations involving this variable.

Since every battery must be either alkaline or not alkaline, the sum of Fiona’s alkaline batteries and her non-alkaline batteries must equal the total number of batteries she owns.

In other words, we can write the following equation:

⇒(numberof alkaline batteries)+(numberof non-alkaline batteries)=totalnumber of batteries⇒A+N=T

Step 3: Solve these equations for the desired variable.

Since the question asks us to determine the number of Fiona’s batteries that are not alkaline, N is the desired variable. So, let’s solve our equation for N:

⇒A+N=T⇒N=T−A

Thus, the number of Fiona’s batteries that are not alkaline can be represented by the expression T – A.

When we check the answer choices, we see that choice E matches the algebraic expression we created.

Important: If none of the answer choices match step 3’s outcome, we must express the outcome differently so that it matches one of the choices. We’ll see how this can occur in the practice examples at the end of this section.

Now let’s examine a second way to answer questions with variables in the answer choices.

Testing numbers

The testing numbers approach to solving questions with variables in the answer choices is as follows:

1. Assign easy-to-work-with numbers to the variables in the question stem.

2. Use the assigned numbers to calculate the value the stem asks us to determine.

3. For each answer choice, replace the variables with the same numbers we assigned in step 1 and calculate the value of that algebraic expression. Eliminate any answer choices that do not have the same value we calculated in step 2. If more than one answer choice remains, repeat the above steps.

The testing numbers approach to solving questions with variables in the answer choices:

1. Assign easy-to-work-with numbers to the variables in the question stem.

2. Use the assigned numbers to calculate the value the stem asks us to determine.

3. For each answer choice, replace the variables with the numbers we assigned in step 1 and calculate the value of that algebraic expression. Eliminate any answer choices that do not have the same value we calculated in step 2. If more than one answer choice remains, repeat the above steps.

Let’s revisit our earlier mini-example and solve it by testing numbers.

Fiona has T batteries, and A of Fiona’s batteries are alkaline batteries. Which of the following represents the number of Fiona’s batteries that are not alkaline?

(A) T × A

(B) A + T

(C) A – T

(D) TA

(E) T – A

Step 1: Assign easy-to-work-with numbers to the variables in the question stem.

The two variables in the stem are T and A, where T is the total number of batteries Fiona owns, and A is the number of alkaline batteries she owns. Thus, when we assign values to T and A, we must do so such that T ≥ N, since the total number of batteries cannot be less than the number of alkaline batteries.

Given this, let’s say that T = 7 and A = 2, which means Fiona owns a total of 7 batteries, and 2 of them are alkaline.

Step 2: Use the assigned numbers to calculate the value the stem asks us to determine.

The stem asks us to determine how many of Fiona’s batteries are not alkaline.

So, if Fiona owns a total of 7 batteries and 2 of them are alkaline, then the number of batteries that are not alkaline is 7 – 2 = 5.

Thus, when we use our assigned values, the answer to the question is, “Fiona owns 5 non-alkaline batteries. ”

Step 3: For each answer choice, replace the variables with the numbers we assigned in step 1 and calculate the value of that algebraic expression. Eliminate any answer choices that do not have the same value we calculated in step 2. If more than one answer choice remains, repeat the above steps.

In steps 1 and 2, we let T = 7 and A = 2, and we used these values to determine that Fiona has 5 non-alkaline batteries. This means we are looking for an answer choice that evaluates to 5 when T = 7 and A = 2.

So, let’s evaluate each answer choice when T = 7 and A = 2:

(A) T × A

If we plug in T = 7 and A = 2, we get T × A = 7 × 2 = 14.

This result tells us that, when T = 7 and A = 2, Fiona will have 14 non-alkaline batteries. Since we already determined that Fiona would have 5 non-alkaline batteries if T = 7 and A = 2, we can eliminate choice A.

(B) A + T

If we plug in T = 7 and A = 2, we get T + A = 7 + 2 = 9.

This result tells us that Fiona will have 9 non-alkaline batteries, which does not align with what we found earlier, that Fiona would have 5 non-alkaline batteries if T = 7 and A = 2. We can eliminate choice B.

(C) A - T

A – T = 2 – 7 = -5

We again did not get the answer of 5 that we’re looking for. Moreover, it’s impossible for Fiona to have -5 non-alkaline batteries; that wouldn’t make sense. We can eliminate choice C.

(D) TA

TA = 72 = 49

This also doesn’t align with Fiona having 5 non-alkaline batteries if T = 7 and A = 2. We can eliminate choice D.

(E) T - A

Plugging in T = 7 and A = 2, we get T – A = 7 – 2 = 5, which matches the result we calculated earlier.

Since we successfully eliminated the other four answer choices, the correct answer must be E.

Let’s practice solving problems using both approaches.

Text Solution:

Let’s first solve the problem algebraically via the three-step process we described earlier:

Step 1: Assign a variable to the value we are asked to determine.

Since we are asked to determine how many cousins Gabe has, let’s let variable g represent the number of cousins Gabe has.

Step 2: Use the given information to create one or more equations involving this variable.

Since we are told that Zara, Fez, and Gabe have a total of t cousins, we can write:

⇒(# of Zara's cousins)+(# of Fez's cousins)+(# of Gabe's cousins)=t⇒z+f+g=t

Step 3: Solve the equation for the desired variable.

Since g, the number of cousins Gabe has, is our desired variable, we will solve the equation for g:

⇒z+f+g=t⇒f+g=t−z⇒g=t−z−f

Since t – f – z and t – z – f are equivalent expressions, the correct answer is D.

Alternate approach:

Now let’s solve the question by testing numbers:

Step 1: Assign easy-to-work-with numbers to the variables in the question stem.

Let’s say Zara has 3 cousins, which means z = 3; let’s say Fez has 7 cousins, which means f = 7; and let’s say t = 16, which means Zara, Fez and Gabe have a total of 16 cousins.

Step 2: Use the assigned numbers to calculate the value the stem asks us to determine.

If Zara and Fez have 3 cousins and 7 cousins, respectively, then Zara and Fez have a total of 10 cousins.

If Zara and Fez have a total of 10 cousins, and Zara, Fez, and Gabe have a total of 16 cousins, then Gabe must have 16 – 10 = 6 cousins.

Thus, when z = 3, f = 7, and t = 16, we find that Gabe has 6 cousins.

So, the correct answer will evaluate to 6 when we let z = 3, f = 7, and t = 16.

Step 3: For each answer choice, replace the variables with the numbers we assigned in step 1 and calculate the value of that algebraic expression. Eliminate any answer choices that do not have the same value we calculated in step 2. If more than one answer choice remains, repeat the above steps.

⇒(A)z+f+t=3+7+16=26

This result tells us that when z = 3, f = 7, and t = 16, Gabe has a total of 26 cousins. Since we already determined that Gabe would have 6 cousins, we can eliminate choice A.

⇒(B)z+f−t=3+7−16=−6

This result tells us that when z = 3, f = 7, and t = 16, Gabe has a total of -6 cousins, which makes no sense. Eliminate choice B.

⇒(C)t−f+z=16−7+3=12

This result tells us that when z = 3, f = 7, and t = 16, Gabe has a total of 12 cousins. Since we already determined that Gabe would have 6 cousins, we can eliminate choice C.

⇒(D)t−f−z=16−7−3=6

This result tells us that when z = 3, f = 7, and t = 16, Gabe has a total of 6 cousins, which matches our result from earlier. Thus, we will keep choice D for now.

⇒(E)z−t−f=3−16−7=−20

This result tells us that when z = 3, f = 7, and t = 16, Gabe has a total of -20 cousins which makes no sense. Eliminate choice E.

By the process of elimination, the correct answer is D.

Correct answer: D

Timoleon, the quality control technician at ABC Corp, inspected T light bulbs yesterday. Of the first F light bulbs Timoleon inspected, one-tenth were defective, and, of the remaining light bulbs, one-twentieth were defective. In terms of T and F, how many defective light bulbs did Timoleon inspect yesterday?

F+T10

F+T15

F+T20

T10+F20

T10−F20

Text Solution:

Let’s first solve the problem algebraically via the three-step process we described earlier:

Step 1: Assign a variable to the value we are asked to determine.

We are asked to determine the number of defective light bulbs Timoleon inspected yesterday. So, let’s let D represent the number of defective light bulbs Timoleon inspected yesterday.

Step 2: Use the given information to create one or more equations involving this variable.

Of the first F light bulbs Timoleon inspected, one-tenth were defective. So, the number of defective light bulbs among the first F light bulbs was 110F=F10.

Next, if there are T light bulbs in total, then after Timoleon inspected F light bulbs, the number of light bulbs remaining to be inspected was T – F.

Since one-twentieth of the remaining light bulbs were defective, the number of defective light bulbs detected in the second round of inspections was 120(T−F)=T20−F20.

Since F10 defective light bulbs were detected in the first round of inspections, and T20−F20 defective light bulbs were detected in the second round of inspections, the sum of these two quantites must equal D, which represents the total number of defective bulbs Timoleon inspected yesterday. Thus, we can write this as an equation representing D in terms of T and F:

⇒D=total number of defective light bulbs Timoleon inspected yesterday⇒D=(#of defective bulbs in first round)+(#of defective bulbs in second round)⇒D=(F10)+(T20−F20)

Step 3: Solve these equations for the desired variable.

Since D is our desired variable, and since the equation D=(F10)+(T20−F20) is already solved for D, we don’t need to solve the equation. However, note that none of the answer choices look exactly like this expression, so we need to use algebraic techniques to simplify this expression until it aligns with an answer choice:

⇒D=(F10)+(T20−F20)⇒D=2F20+T20−F20⇒D=F20+T20⇒D=F+T20

Now we know that D=F+T20, which aligns with the expression in answer choice C.

Alternate approach:

Now let’s solve the question by testing numbers:

Step 1: Assign easy-to-work-with numbers to the variables in the question stem.

Let’s say T = 50, and F = 30.

Note: These values of T and F correspond to Timoleon inspecting a total of 50 light bulbs, including 30 light bulbs in the first round. We chose these values because this means he inspected 50 – 30 = 20 light bulbs in the second round. Note that it’s easy to find one-tenth of 30 and one-twentieth of 20, which we’ll be required to do by the language in the question.

Step 2: Use the assigned numbers to calculate the value the stem asks us to determine.

In the first round, Timoleon inspected 30 light bulbs, and one-tenth were defective, which means the number of defective light bulbs inspected in the first round was 110of 30=110×30=3010=3.

In the second round, Timoleon inspected 20 light bulbs, and one- twentieth were defective, which means the number of defective light bulbs inspected in the second round was 120of 20=120×20=2020=1.

Thus, when T = 50 and F = 30, we find that Timoleon inspected a total of 3 + 1 = 4 defective light bulbs yesterday.

So, the correct answer will evaluate to 4 when we let T = 50 and F = 30.

Step 3: For each answer choice, replace the variables with the numbers we assigned in step 1 and calculate the value of that algebraic expression. Eliminate any answer choices that do not have the same value we calculated in step 2. If more than one answer choice remains, repeat the above steps.

⇒(A)F+T10=30+5010=8010=8

This result tells us that when T = 50 and F = 30, Timoleon inspected a total of 8 defective light bulbs. Since we already determined that Timoleon would have inspected a total of 4 defective light bulbs, we can eliminate choice A.

⇒(B)F+T15=30+5015=8015=163=513

This result tells us that when T = 50 and F = 30, Timoleon inspected a total of 513 defective light bulbs, which doesn’t even make sense. Eliminate choice B.

⇒(C)F+T20=30+5020=8020=4

This result tells us that when T = 50 and F = 30, Timoleon inspected a total of 4 defective light bulbs, which matches our result from earlier. Thus, we will keep choice C for now.

⇒(D)T10+F20=5010+3020=5+112=612

This result tells us that when T = 50 and F = 30, Timoleon inspected a total of 612 defective light bulbs, which doesn’t make sense. Eliminate choice D.

⇒(E)T10−F20=5010−3020=5−112=312

This is another answer that doesn’t make sense. Eliminate choice E.

By the process of elimination, the correct answer is C.

Correct answer: C

Coordinate Geometry is Currently Tested on the GMAT
Updated on: Jul 2025

While most geometrical concepts were removed from the GMAT Focus Edition, which GMAC now refers to simply as "the GMAT" once again, Coordinate Geometry is still tested.

In addition, some basic geometry concepts, including calculating the area and perimeter of shapes like squares, triangles, and circles, are still tested on the current GMAT. However, whenever a geometry formula outside of coordinate geometry is required, it will be provided in the question stem.

About Hard Tests

The questions in this Hard Chapter Test are as difficult as any that appear on the GMAT. So, at this stage of your preparation, it does not make sense to seek answer them in under two minutes.

Rather, for best results, you should hide the timer and answer these questions untimed, taking as much time as you need to fully understand and analyze each question.

By practicing in this way, you will give yourself time to learn to see what’s going on in the questions and to learn to work your way to correct answers to difficult questions.
