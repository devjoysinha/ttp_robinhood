4. Properties of Numbers

4.6.9 Drawing Conclusions Involving Inequalities

Some GMAT questions ask us to draw conclusions involving inequalities based on information about the signs of the variables involved.

Consider the following mini-example:

If k > 0, j ≠ 0, and k > j, which of the following must be true?

(A) j + k > 0           (B) k – j > 0          (C) jk > 0

To answer this question, we must first determine all the possible cases that satisfy the given information by focusing on whether each variable can be negative or positive.

First, since the question says k > 0, we know k is positive.

Next, we know j ≠ 0, which means j cannot be 0. We also know k > j (k is greater than j), but this does not tell us whether j is positive or negative. The positive number k could be greater than j in either situation. This means there are two possible cases regarding the signs of j and k:

So, since we know k is greater than j, we placed k to the right of j in the table above. Thus, when we refer to case i, we can quickly see that j and k are both positive, and that k is greater than j.

Now that we have identified all possible cases, we can analyze each of the three given options:

(A) j + k > 0

If case i is true, then the inequality j + k > 0 becomes (+) + (+) > 0. Since the sum of two positive numbers is always positive, we can conclude that option A must be true for case i.

If case ii is true, then the inequality j + k > 0 becomes ( – ) + (+) > 0. Since the sum of a negative number and a positive number can be negative (e,g. , -5 + 3 = -2), positive (e,g. , -5 + 8 = 3), or zero (e,g. , -5 + 5 = 0), we can conclude that option A may or may not be true for case ii.

Since we have now shown that option A need not be true for all cases, we can move on to the next option.

(B) k – j > 0

Since it is given that k is greater than j, this means k – j must be greater than 0. So, we can conclude that option B (k – j > 0) must be true.

As we have just seen, we can sometimes draw conclusions without needing to consult the possible cases.

(C) jk > 0

If case i is true, then the inequality jk > 0 becomes (+)(+) > 0. Since the product of two positive numbers is always positive, we can conclude that option C must be true for case i.

If case ii is true, then the inequality jk > 0 becomes (+)( – ) > 0. Since the product of a positive number and a negative number is always negative, we can conclude that option C is not true for case ii.

We have now shown that option C (jk > 0) need not be true for all cases.

Thus, only option B must be true.

We can generalize the above technique as follows:

When we must prove or disprove inequalities based on given information about the signs of certain variables, we can follow this two-step technique:

• Step 1: Identify all cases that satisfy the given information.

• Step 2: For each statement, test all cases identified in step 1.

Don’t forget to factor

In some cases, the given information and/or the answer choices will include algebraic expressions with sums or differences. In such cases, we must remember that we can factor the expressions, and doing so may help us make additional conclusions. Consider the following mini-example:

If x < 0 and x2y + 2xy > 0, which of the following must be true?

(A) x + y > 0           (B) y – x > 0           (C) xy + 2y < 0

Before we apply our 2-step technique, we can first factor x2y + 2xy > 0 as follows:

⇒ x 2 y+2xy>0 ⇒xy x+2 >0 ⇒ x x+2 y >0

Please note that we rearranged the terms so that x and (x + 2) are next to each other in the product (x)(x + 2)(y). Doing so will help to remind us that (x + 2) is greater than x, and thus (x + 2) will be placed to the right of x when we create our table of possible cases.

Also note that the given inequality (x)(x + 2)(y) > 0 indirectly tells us that x ≠ 0, x + 2 ≠ 0, and y ≠ 0; otherwise, the given inequality cannot be true.

Now, let’s apply the technique.

Step 1: Identify all cases that satisfy the given information.

We are told that x is negative, which means x + 2 can be either negative or positive. For example, if x = -5, then x + 2 = -5 + 2 = -3, which is negative. On the other hand, if x = -0.3, then x + 2 = -0.3 + 2 = 1.7, which is positive.

Since we don’t have any direct information about y, let’s add the two cases we know about to create the following table. We don’t know whether y is greater or less than x or x + 2, but we’ll put it to the right for now:

For case ii, we can replace x with “ – “ and replace (x + 2) with “+“ to get ( – )(+)(y) > 0, in which case we can conclude that y is negative. Let’s add this information to our table:

Now that we have identified two possible cases, we can analyze each answer choice.

Step 2: For each statement, test all cases identified in step 1.

(A) x + y < 0

If case i is true, then the inequality x + y < 0 becomes ( – ) + (+) < 0. Since the sum of a negative number and a positive number can be negative, positive, or zero, we can conclude that option A is not necessarily true for case i. So, we can stop analyzing option A and conclude that it need not be true.

(B) y – x > 0

If case i is true, then the inequality y – x > 0 becomes (+) – ( – ) > 0. Since a positive number minus a negative number always results in a positive number, we can conclude that option B is true for case i.

If case ii is true, then the inequality y – x > 0 becomes ( – ) – ( – ) > 0. Since a negative number minus a negative number can result in a positive number, a negative number, or zero, we can conclude that option B is not necessarily true for case ii. Option B need not be true.

(C) xy + 2y < 0

Before checking both cases for option C, let’s first factor option C: (y)(x + 2) < 0.

If case i is true, then the inequality (y)(x + 2) < 0 becomes (+)( – ) < 0. Since the product of positive number and a negative number is always negative, we can conclude that option C is true for case i.

If case ii is true, then the inequality (y)(x + 2) < 0 becomes ( – )(+) < 0. Since the product of negative number and a positive number is always negative, we can conclude that option C is true for case ii.

Since option C is true for both possible cases, we can conclude that option C MUST be true.

Step 2: For each statement, test all cases identified in step 1.

Now that we have identified the possible cases, we can analyze each statement.

I. jc – c2 < 0

First, we can factor statement I: c(j – c) < 0.

From here, our table tells us that c is negative in both cases. We are also told in the question that j is greater than c, which means j – c will be greater than 0. In other words, the given information indirectly tells us that j – c is positive.

So, if either case i or case ii is true, the rephrased statement c(j – c) < 0 becomes ( – )(+) < 0. Since the product of a negative number and a positive number is always negative, we can conclude that statement I must be true.

II. j2 – c2 < 0

Let’s first factor the difference of squares in statement II: (j + c)(j – c) < 0.

Since we already concluded that j – c is positive, we can replace (j – c) with a “+” symbol as we examine our two possible cases.

If case i is true, then the inequality (j + c)(j – c) < 0 becomes [( – ) + ( – )](+) < 0, and since the sum of two negative numbers is always negative, we can simplify this inequality to ( – )(+) < 0. Since the product of negative number and a positive number is always negative, we can conclude that statement II is true for case i.

If case ii is true, then the inequality (j + c)(j – c) < 0 becomes [(+) + ( – )](+) < 0. From here, since the sum of a positive number and a negative number can be positive, negative, or zero, statement II is not necessarily true for case ii. Thus, statement II need not be true.

III. c3 – c2j < 0

Let’s first factor statement III: c2(c – j) < 0.

Because we know from the question that j > c, this means that c – j must be negative.

We also know that c is negative, and when you square any negative number, the result is positive, which means c2 must be positive.

Thus, even without considering any cases, we know that the inequality c2(c – j) < 0 becomes (+)( – ) < 0, and since the product of a positive number and a negative number is always negative, we can conclude that statement III must be true.

Of the three statements, only I and III must be true.
