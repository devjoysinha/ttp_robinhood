13. Overlapping Sets

13.2.5 Solving for a Combination of Cells in an Overlapping Set

Thus far in this chapter, we have solved overlapping sets questions in which we needed to determine the value in a single cell from an overlapping set-matrix. However, we can be asked to determine the value of multiple cells in an overlapping set-matrix.

Must Know:
Toggle bookmark

We must be prepared to solve for multiple cells in an overlapping set problem.

For example, what if we are given two sets and asked for the number of elements that belong to exactly 1 group? That is, we’re being asked for the total number of items that belong to Group A but not Group B plus the total number in Group B but not Group A.

Let’s highlight those cells in the sample table below.

If we let the number of people in Group A only = x and the number of people in Group B only = y, then we need to determine the sum of x and y.

We can solve this problem type by applying the overlapping sets formula we introduced earlier in the chapter. Sticking to the same example we would have:

A Only + B Only + (Both A and B) + (Neither A nor B) = Total

Substituting our variables, we have:

x + y + (Both A and B) + (Neither A nor B) = Total

x + y = Total – (Both A and B) – (Neither A nor B)

Thus, if we have values for the total, both, and neither terms, we can determine the sum of x and y.

Must Know:
Toggle bookmark

If we have values for the total, both, and neither terms in an overlapping set, we can determine the number of elements that belong to exactly one group.

Let’s now practice this idea with a mini-example.

At a particular school, only French and German are offered by the foreign language department. In a math class of 80 students, 30 are taking both French and German, and 25 are taking neither French nor German. How many students are taking exactly one language?

From the set matrix, we see that this is the quantity (x + y).

Important: From the given information, we cannot determine the individual values of x and y. However, since our goal is to calculate the sum, x + y, we need only recognize that the sum of the four main values in the set-matrix—30, x, y, and 25—must equal the total number of students.

In other words, we can write 30 + x + y + 25 = 80, and then solve for the sum x + y.

⇒ 30 + x + y + 25 = 80

⇒ x + y + 55 = 80

⇒ x + y = 25

This, in essence, is the same as applying overlapping sets formula: A only + B only + Both + Neither = Total

If we let A = French and B = German, then we can rewrite the formula as:

⇒ French only + German only + Both + Neither = Total

Now we can substitute the known information from the set matrix:

⇒ x + y + 30 + 25 = 80 (Notice that we are, once again, using the fact that the four main boxes in the set-matrix must add to the total number of students)

⇒ x + y + 55 = 80

⇒ x + y = 25

Thus, we know that the number of students taking exactly one language is 25.

Must Know:
Toggle bookmark

We can solve for multiple cells in an overlapping set problem using a set-matrix or the overlapping sets formula: Total = A only + B only + Both + Neither.

Thus far, in this section, we have focused on determining the sum of two cells of a set-matrix. More specifically, we have calculated the sum “A Only + B Only. ” However, we can use the same techniques to solve for the sum of three cells in a set-matrix.

Consider the following mini-example

If there are 30 campers, and 7 of them like neither fishing nor swimming, how many campers like at least one of those two activities?

Let’s highlight the cells representing campers who like at least one the two activities:

 

Like Swimming

Do NOT Like Swimming

Total

Like Fishing

x

z

  

Do Not Like Fishing

y

7

  

Total

    

30

If we let x = the number of campers who like swimming and fishing, let y = the number of campers who like swimming but not fishing, and let z = the number of campers who like fishing but not swimming, then we want to determine the sum x + y + z.

To do so, we can recognize that the values in the four main boxes—x, y, z, and 7—must add to the total number of campers.

In other words, x + y + z + 7 = 30, which means x + y + z = 23, which means 23 campers like at least one activity.

So, although it’s impossible to determine the individual values of x, y and z, we can find the sum x + y + z.

Notice that this solution is almost identical to applying the overlapping sets formula.

If A = Swimming and B = Fishing, then our solution becomes:

⇒ Swimming only + Fishing only + Both + Neither = Total

⇒ y + z + x + 7 = 30

⇒ y + z + x = 23

Once again, we can conclude that 23 campers like at least one activity.

In general, if we know the total population of a group and at least one of the four main values (A Only, B Only, Both A and B, or Neither A nor B), then we can find the sum of the other three values.

Coordinate Geometry is Currently Tested on the GMAT
Updated on: Jul 2025

While most geometrical concepts were removed from the GMAT Focus Edition, which GMAC now refers to simply as "the GMAT" once again, Coordinate Geometry is still tested.

In addition, some basic geometry concepts, including calculating the area and perimeter of shapes like squares, triangles, and circles, are still tested on the current GMAT. However, whenever a geometry formula outside of coordinate geometry is required, it will be provided in the question stem.
