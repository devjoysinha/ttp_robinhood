19. Functions and Sequences

19.1 Introduction

This chapter reviews functions and sequences as well as the rules that apply to them.

Let’s begin with the study of functions.

Functions

We can use function notation to define and evaluate algebraic expressions involving one variable.

Typical function notation looks like the following:

If f(x) = x³ + 2x – 1, what is the value of f(5)?

The first expression “f(x)” is read “f of x”, and it means a “function of x”.

Similarly, “f(5)” is read “f of 5.”

Notice that in f(5), the 5 is in the place that x occupies in f(x). This hints at how we answer this question.

To find the value of f(5), we must plug 5 in for x in the given expression for f(x):

f(x)  = x³ + 2x – 1
f(5)  = (5)³ + 2×5 – 1
      = 125 + 10 – 1
      = 134

Notice that this work is very similar to how we might answer:

What is the value of x³ + 2x – 1 when x = 5?

This is essentially the same question written in a new way. Here are four equivalent ways of asking:

  • What is the value of x³ + 2x – 1 when x = 5?
  • If y = x³ + 2x – 1, what is the value of y when x = 5?
  • If f(x) = x³ + 2x – 1, what is the value of f(5)?
  • If function f is defined as f(x) = x³ + 2x – 1, what is the value of f when x = 5?

We can answer all these by taking the original expression, replacing x with 5, and calculating the result.

Must Know: Evaluating a function is very similar to evaluating an expression.

The numbers a function uses are called inputs, the numbers it produces are called outputs, and the function is like a machine that takes us from inputs to outputs.

Thus, in the above example, we see that when we input 5, the output is 134.

Using function terminology, we say, “134 is the value of f at x = 5.” This is just another way to say, “The algebraic expression x³ + 2x – 1 has a value of 134 when x = 5.”

Must Know: We can think of a function as a math machine that uses input numbers to produce output numbers.

Practice Example 1

If f(x) = x³ / x² – √x, what is the value of f(4)?

  • A) 4
  • B) 2 (correct)
  • C) 0
  • D) -2
  • E) -4
Solution

To calculate f(4):

  f(4)  = 4³ / 4² – √4
        = 64 / 16 – 2
        = 4 – 2
        = 2

Practice Example 2

If f(x) = 28 – 2x and k is a constant such that f(k) = 10, what is the value of k?

  • A) 8
  • B) 9 (correct)
  • C) 101
  • D) 11
  • E) 12
Solution

f(k) = 10 ⇒ 28 – 2k = 10
2k = 28 – 10 = 18 ⇒ k = 9
