6. Inequalities and Absolute Values

6.18 Solution Set Questions

Some algebra questions will ask us to determine the solution set for an inequality, which means to describe all the solutions of the inequality.

Must Know:
Toggle bookmark

The solution set of an inequality describes all the solutions of the inequality.

To see how this works in practice, let’s consider the following mini-example:

What is the solution set of the inequality 2x – 5 < 7?

One way to solve this problem is to apply algebraic techniques to isolate the variable x, just like we would with an equation:

⇒2x−5<7 ⇒2x<12 ⇒x< 12 2 ⇒x<6

This tells us that every value of x that is less than 6 will be a solution to the original inequality, 2x – 5 < 7. In other words, x < 6 is the solution set of the inequality 2x – 5 < 7.

We can also display this solution set on a number line. So, we might say that the graph below indicates the solution set of the inequality 2x – 5 < 7, as the shaded arrow indicates all the numbers that are less than 6. Note that the circle over 6 is not filled in because 6 is not a solution to this inequality. (After all, 6 is not less than 6. )

Before we explore a second way to solve solution set questions, let’s first examine the various ways the test-makers can present this same type of question.

All of the examples below are asking the same question but in different ways:

Example 1:

Which of the following specifies the set of all numbers x such that 2x – 5 < 7?

(A) x > 6

(B) x < 6

(C) x ≤ 6

(D) x ≥ 6

(E) -6 < x < 6

Example 2:

The solution set of the inequality 2x – 5 < 7 is the set of all real numbers x such that

(A) x > 6

(B) x < 6

(C) x ≤ 6

(D) x ≥ 6

(E) -6 < x < 6

Example 3:

Which of the following is equivalent to the inequality 2x – 5 < 7?

(A) x > 6

(B) x < 6

(C) x ≤ 6

(D) x ≥ 6

(E) -6 < x < 6

Example 4:

Which of the following describes all values of x for which 2x – 5 < 7?

(A) x > 6

(B) x < 6

(C) x ≤ 6

(D) x ≥ 6

(E) -6 < x < 6

Example 5:

On the number line, which of the following describes the set of all numbers x such that 2x – 5 < 7?

(A) x > 6

(B) x < 6

(C) x ≤ 6

(D) x ≥ 6

(E) -6 < x < 6

While the language in the above examples varies, in all of them we are looking for the solution set for some given inequality.

Another way to ask solution set questions is to show the solution set on a number line and ask which inequality among the answer choices has a solution set that matches that shown on the line. Here’s an example of such a question:

Example 6:

Which of the following inequalities is an algebraic expression for the shaded part of the number line shown above?

(A) 2x – 5 < 7

(B) 2x – 5 > 7

(C) 2x + 7 < 5

(D) 5x – 7 > 2

(E) 7x – 5 < 2

Here, the answer is A, which we could determine by solving all the answer choices until we found the inequality with the solution set x < 6.

Now that we know how to identify solution set questions, let’s examine two ways to solve them:

1. Applying algebraic techniques

2. Testing numbers

Must Know:
Toggle bookmark

Solution set questions can be solved by applying algebraic techniques or by testing numbers.

Let’s explore these two approaches with a new mini-example:

Which of the following describes all values of x for which 2(3x – 4) < 5(x – 1)?

(A) x > 3

(B) x < 1

(C) x < -1

(D) x < 3

(E) x > 4

Applying algebraic techniques

As we saw in the previous mini-example, we can apply algebraic techniques to solve solution set questions. In this new mini-example, the algebra will just be a bit more involved:

⇒2 3x−4 <5 x−1 ⇒6x−8<5x−5 ⇒x−8<−5 ⇒x<3

Since our calculations resulted in the conclusion that x is less than 3, the correct answer must be D.

Now let’s solve the same mini-example via the second strategy.

Testing numbers

Which of the following describes all values of x for which 2(3x – 4) < 5(x – 1)?

(A) x > 3

(B) x < 1

(C) x < -1

(D) x < 3

(E) x > 4

To begin, let’s make sure we understand what the answer choices are telling us.

For example, answer choice A (x > 3) tells us that every value of x that’s greater than 3 will be a solution to the inequality 2(3x – 4) < 5(x – 1). Choice A also tells us that every value of x that’s less than or equal to 3 will not be a solution to the inequality 2(3x – 4) < 5(x – 1).

Similarly, answer choice B (x < 1) tells us that every value of x that’s less than 1 will be a solution to the inequality and that every value of x that’s greater than or equal to 1 will not be a solution to the inequality.

Since these concepts apply to all five answer choices in a solution set question, we can eliminate answer choices by testing values of x to see whether the same values satisfy both those answer choices and the original inequality.

In general, one of the easiest numbers to test is x = 0. So, let’s replace x with 0 to test whether 0 is a solution to the given inequality:

⇒Is 2 3x−4 <5 x−1 ? ⇒Is 2 3 0 −4 <5 0−1 ? ⇒Is 2 −4 <5 −1 ? ⇒Is −8<−5? ⇒Yes.

Since x = 0 satisfies the given inequality, 0 is a solution to this inequality. This means we can eliminate every answer choice that indicates 0 is not a solution, and we will keep (for now) every answer choice that includes 0 as a solution. We can test this by replacing x with 0 in each answer choice:

Choice A

⇒Is x>3? ⇒Is 0>3? ⇒No.

Because 0 does not satisfy the inequality x > 3, this indicates 0 is not a solution of this inequality. So, we can eliminate choice A.

Choice B

⇒Is x<1? ⇒Is 0<1? ⇒Yes.

Because 0 satisfies the inequality x < 1, this indicates 0 is a solution of this inequality. So, we can keep choice B for now, as it agrees with the original inequality on 0 being a solution.

Choice C

⇒Is x<−1? ⇒Is 0<−1? ⇒No.

We see that 0 is not a solution of this inequality. So, we can eliminate choice C.

Choice D

⇒Is x<3? ⇒Is 0<3? ⇒Yes.

This indicates 0 is a solution of this inequality. So, we can keep choice D for now.

Choice E

⇒Is x>4? ⇒Is 0>4? ⇒No.

We find that 0 is not a solution of this inequality. So, we can eliminate choice E.

So, after testing one x-value, only answer choices B (x < 1) and D (x < 3) remain.

Important: At this point, we must avoid testing irrelevant numbers. That is, we must avoid testing x-values that will not allow us to eliminate either of the remaining answer choices.

For example, testing x = -10 would not help because -10 is less than 1 and less that 3, which means the two remaining answers, B and D, will both tell us that -10 is a solution. So, instead, we want to test an x-value that is a solution for only one of the two remaining answer choices.

For example, testing x = 2 would be useful since answer choice B (x < 1) tells us x = 2 is not a solution, while answer choice D (x < 3) tells us x = 2 is a solution:

Choice B                 Choice D

⇒Is x<1? ⇒Is 2<1? ⇒No.             ⇒Is x<3? ⇒Is 2<3? ⇒Yes.

Thus, testing x = 2 will allow us to eliminate one of the remaining answer choices. So, let’s see if x = 2 is a solution to the original inequality in the question by replacing x with 2:

⇒Is 2 3x−4 <5 x−1 ? ⇒Is 2 3 2 −4 <5 2−1 ? ⇒Is 2 2 <5 1 ? ⇒Is 4<5? ⇒Yes.

Since x = 2 satisfies the given inequality, 2 is a solution. This means we can eliminate any answer choice that says 2 is not a solution. In other words, we can eliminate choice B and keep choice D based on our work above.

So, by process of elimination, the correct answer is D.

Must Know:
Toggle bookmark

Avoid testing numbers that will not allow us to eliminate any answer choices.

Now that we’ve explored both strategies in this one example, we may be wondering, when we approach a new question involving solution sets…

Which strategy should be applied to a solution set question?

If the given inequality is somewhat straightforward, then applying algebraic techniques will likely be the faster method. On the other hand, if the given inequality is complex, testing numbers will likely be faster.

Must Know:
Toggle bookmark

If the given inequality is straightforward, then applying algebraic techniques will likely be the faster method. If the given inequality is complex, testing numbers will likely be faster.

Let’s see how we do with a much harder mini-example:

Which of the following describes all values of x for which 2x−1 + x+1 <6 ?

(A) -5 < x < -1

(B) -2 < x < 2

(C) x < -4 or x > 2

(D) x < -3

(E) x > -1

Because this inequality looks fairly complex and difficult to solve via algebraic techniques, we’ll try testing numbers.

At a quick glance, we might be able to see that some answer choices indicate that x = 0 is a solution and other choices indicate x = 0 is not a solution, so let’s see if x = 0 satisfies the given inequality:

⇒Is 2x−1 + x+1 <6? ⇒Is 2 0 −1 + 0+1 <6? ⇒Is −1 + 1 <6? ⇒Is 1+1<6? ⇒Is 2<6? ⇒Yes.

Since x = 0 satisfies the given inequality, 0 is a solution. This means we can eliminate every answer choice that says 0 is not a solution and keep (for now) every answer choice that includes 0 as a solution:

Choice A

⇒Is −5<x<−1? ⇒Is −5<0<−1? ⇒No.

Because 0 is not between -5 and -1, it is not a solution of this inequality. So, we can eliminate choice A.

Choice B

⇒Is −2<x<2? ⇒Is −2<0<2? ⇒Yes.

In this case, 0 is between -2 and 2, which means 0 is a solution of this inequality. So, we can keep choice B for now.

Choice C

⇒Is x<−4 or is x>2? ⇒Is 0<−4 or is 0>2? ⇒No.

Here we have an inequality with an “or”, which means that 0 will be a solution if it satisfies either of the two given inequalities. However, in this case, we see that 0 is not less than -4, nor is it greater than 2. This means 0 is not a solution of this inequality. So, we can eliminate choice C.

Choice D

⇒Is x<−3? ⇒Is 0<−3? ⇒No.

This indicates 0 is not a solution of this inequality. So, we can eliminate choice D.

Choice E

⇒Is x>−1? ⇒Is 0>−1? ⇒Yes.

We find that 0 is a solution of this inequality. So, we can keep choice E for now.

At this point, answer choices B (-2 < x < 2) and E (x > -1) remain.

From here, we want to test an x-value that is a solution to only one of the two remaining answer choices.

Notice that testing x = 3 would be useful since answer choice B (-2 < x < 2) indicates that 3 is not a solution, and answer choice E (x > -1) indicates that 3 is a solution:

Choice B                             Choice E

⇒Is −2<x<2? ⇒Is −2<3<2? ⇒No.                ⇒Is 3>−1? ⇒Is 3>−1? ⇒Yes.

So, let’s test x = 3 in the original inequality:

⇒Is 2x−1 + x+1 <6? ⇒Is 2 3 −1 + 3+1 <6? ⇒Is 5 + 4 <6? ⇒Is 5+4<6? ⇒Is 9<6? ⇒No.

Since x = 3 does not satisfy the given inequality, 3 is not a solution, which means we can eliminate any answer choice that indicates 3 is a solution. So, we will keep choice B and eliminate choice E.

By process of elimination, the correct answer is B.

Text Solution:

To solve the problem via algebraic techniques, let’s start by isolating x2.

⇒16≥25− x 2 ⇒16+ x 2 ≥25 ⇒ x 2 ≥9

To isolate the variable x, we will take the square root of both sides, which gives us the following:

⇒ x 2 ≥9 ⇒ x 2 ≥ 9 ⇒ x ≥3

Now that we have the absolute value of x, so we can follow our absolute value rules to determine the solution set:

Case 1: x is positive

⇒x≥3

Case 2: x is negative

⇒−x≥3 ⇒x≤−3

Thus, x ≥ 3 or x ≤ – 3.

The Answer is C

Alternate solution:

Now let’s solve the problem by testing values for x.

Since some answer choices indicate that x = 0 is a solution while others indicate x = 0 is not a solution, let’s see if x = 0 satisfies the given inequality:

⇒Is16≥25-x 2? ⇒Is16≥25-0 2? ⇒Is16≥25? ⇒No.

Since x = 0 does not satisfy the given inequality, 0 is not a solution. This means we can eliminate every answer choice that says 0 is a solution, and we will keep (for now) every answer choice that says 0 is not a solution:

⇒ Eliminate choice A since -4 ≤ x ≤ 4 indicates that x = 0 is a solution, as 0 is between -4 and 4.

⇒ Eliminate choice B since -3 ≤ x ≤ 3 indicates that x = 0 is a solution, as 0 is between -3 and 3.

⇒ Keep choice C since x ≤ -3 or x ≥ 3 indicates that x = 0 is not a solution, as 0 is neither less than or equal to -3 nor greater than or equal to 3.

⇒ Keep choice D since x ≤ -4 or x ≥ 4 indicates that x = 0 is not a solution. As with choice C, 0 is between -4 and 4, not less than or equal to -4 or greater than or equal to 4.

⇒ Keep choice E since x ≥ 3 indicates that x = 0 is not a solution, as 0 is not greater than or equal to 3.

We are down to choices C, D and E.

From here, we will test another x-value. Note that choice C indicates that -3 is a solution, since -3 is less than or equal to -3. For choices D and E, meanwhile, -3 is not a solution, because -3 is not less than or equal to -4, nor is it greater than or equal to 3 or 4. So, let’s test whether -3 is a solution to the inequality:

⇒Is 16≥25− x 2 ? ⇒Is 16≥25− −3 2 ? ⇒Is 16≥25−9? ⇒Is 16≥16? ⇒Yes.

Note that 16 is indeed greater than or equal to 16, because it is equal. Since -3 is a solution, we can eliminate answer choices D and E since they both indicate that -3 is not a solution.

So, by process of elimination, the correct answer is C.

Correct answer: C

This inequality could be hard to manipulate algebraically, so let’s solve the question by testing values of x, beginning with x = 0:

⇒Is x+2 x−3 <−4? ⇒Is 0+2 0−3 <−4? ⇒Is − 2 3 <−4? ⇒No.

Since x = 0 does not satisfy the given inequality, 0 is not a solution. This means we can eliminate every answer choice that says 0 is a solution, and we will keep (for now) every answer choice that says 0 is not a solution:

⇒ Eliminate choice A since -3 < x < 2 indicates that x = 0 is a solution, as 0 is between -3 and 2.

⇒ Keep choice B since 2 < x < 3 indicates that x = 0 is not a solution, as 0 is not between 2 and 3.

⇒ Eliminate choice C since -2 < x < 3 indicates that x = 0 is a solution, as 0 is between -2 and 3.

⇒ Keep choice D since -3 < x < -2 indicates that x = 0 is not a solution, as 0 is not between -3 and -2.

⇒ Keep choice E since x > 2 indicates that x = 0 is not a solution, as 0 is not greater than 2.

We are down to choices B, D and E.

From here, we will test another x-value. Note that a couple of the answer choices involve ranges that are bounded by consecutive integers (like 2 < x < 3). To test whether these ranges are correct, we may need to look at decimals between these integers. With that in mind, let’s test whether x = 2.5 is a solution to the inequality:

⇒Is x+2 x−3 <−4? ⇒Is 2.5+2 2.5−3 <−4? ⇒Is 4.5 −0.5 <−4? ⇒Is − 4.5 0.5 <−4? ⇒Is − 45 5 <−4? ⇒Is −9<−4? ⇒Yes.

Since 2.5 is a solution, we can eliminate answer choice D since -3 < x < -2 indicates that 2.5 is not a solution, because 2.5 is not between -3 and -2.

Since choices B (2 < x < 3) and E (x > 2) both indicate that 2.5 is a solution, we must keep these choices and test a third x-value.

Let’s test whether 4 is a solution to the inequality:

⇒Is x+2 x−3 <−4? ⇒Is 4+2 4−3 <−4? ⇒Is 6 1 <−4? ⇒Is 6<−4? ⇒No.

Since 4 is not a solution, we can eliminate choice E, because 4 is greater than 2, which means 4 is a solution to the inequality x > 2.

So, by process of elimination, the correct answer is B.

Text Solution:

The graph on the number line tells us that all values of x from -3 to 4, inclusive, are the solutions to one of the five answer choices.

So, one approach would be to solve each of the five answer choices until we find the inequality with the solution set -3 ≤ x ≤ 4.

A faster approach is to test values of x and eliminate answer choices. So, let’s do that.

Since we are told -3 ≤ x ≤ 4 is the solution set for some inequality, let’s test one of the extreme values in this range, perhaps x = -3.

Since -3 is part of the given solution set, we will eliminate any answer choices for which -3 is not a solution, and we will keep (for now) any answer choices for which -3 is a solution.

(A) |x + 2| + |x – 3| ≤ 7

⇒Is x+2 + x−3 ≤7? ⇒Is −3 +2 + −3 −3 ≤7? ⇒Is −1 + −6 ≤7? ⇒Is 1+6≤7? ⇒Is 7≤7? ⇒Yes.

Since -3 is a solution to the inequality, we will keep choice A.

(B) |x – 2| + |x + 3| ≤ 7

⇒Is x−2 + x+3 ≤7? ⇒Is −3 −2 + −3 +3 ≤7? ⇒Is −5 + 0 ≤7? ⇒Is 5+0≤7? ⇒Is 5≤7? ⇒Yes.

Since -3 is a solution to the inequality, we will keep choice B.

(C) |x – 3| + |x – 4| ≤ 7

⇒Is x−3 + x−4 ≤7? ⇒Is −3 −3 + −3 −4 ≤7? ⇒Is −6 + −7 ≤7? ⇒Is 6+7≤7? ⇒Is 13≤7? ⇒No.

Since -3 is not a solution to the inequality, we will eliminate choice C.

(D) |x + 2| + |x – 5| ≤ 7

⇒Is x+2 + x−5 ≤7? ⇒Is −3 +2 + −3 −5 ≤7? ⇒Is −1 + −8 ≤7? ⇒Is 1+8≤7? ⇒Is 9≤7? ⇒No.

Since -3 is not a solution to the inequality, we will eliminate choice D.

(E) |x – 3| + |x – 2| ≤ 7

⇒Is x−3 + x−2 ≤7? ⇒Is −3 −3 + −3 −2 ≤7? ⇒Is −6 + −5 ≤7? ⇒Is 6+5≤7? ⇒Is 11≤7? ⇒No.

Since -3 is not a solution to the inequality, we will eliminate choice E.

We have narrowed the answer choices down to A and B.

Now let’s test the other extreme value in the solution set -3 ≤ x ≤ 4, meaning x = 4:

(A) |x + 2| + |x – 3| ≤ 7

⇒Is x+2 + x−3 ≤7? ⇒Is 4 +2 + 4 −3 ≤7? ⇒Is 6 + 1 ≤7? ⇒Is 6+1≤7? ⇒Is 7≤7? ⇒Yes.

Since 4 is a solution to the inequality, we will keep choice A.

(B) |x – 2| + |x + 3| ≤ 7

⇒Is x−2 + x+3 ≤7? ⇒Is 4 −2 + 4 +3 ≤7? ⇒Is 2 + 7 ≤7? ⇒Is 2+7≤7? ⇒Is 9≤7? ⇒No.

Since 4 is not a solution to the inequality, we will eliminate choice B.

By the process of elimination, the correct answer is A.

Coordinate Geometry is Currently Tested on the GMAT
Updated on: Jul 2025

While most geometrical concepts were removed from the GMAT Focus Edition, which GMAC now refers to simply as "the GMAT" once again, Coordinate Geometry is still tested.

In addition, some basic geometry concepts, including calculating the area and perimeter of shapes like squares, triangles, and circles, are still tested on the current GMAT. However, whenever a geometry formula outside of coordinate geometry is required, it will be provided in the question stem.

About Hard Tests

The questions in this Hard Chapter Test are as difficult as any that appear on the GMAT. So, at this stage of your preparation, it does not make sense to seek answer them in under two minutes.

Rather, for best results, you should hide the timer and answer these questions untimed, taking as much time as you need to fully understand and analyze each question.

By practicing in this way, you will give yourself time to learn to see what’s going on in the questions and to learn to work your way to correct answers to difficult questions.
